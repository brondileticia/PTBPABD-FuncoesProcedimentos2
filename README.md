# Atividade - Procedure Salary Histogram

## 📚 Descrição

Esta atividade consiste na criação de um procedimento armazenado que gera um histograma de salários dos instrutores.

## 🎯 Objetivo

Criar a procedure `salaryHistogram` que:
- Recebe o número de intervalos como parâmetro
- Distribui os salários dos instrutores em intervalos
- Retorna uma tabela com: intervalo, valor mínimo, valor máximo e total
